## Overview

Ghost is a web proxy frontend / internet browsing hub designed for bypassing internet censorship. It includes browser-style proxying, built-in apps/games, and customization features, built with [React](https://github.com/facebook/react).

> [!WARNING]
> This Ghost fork is currently unstable. Expect regressions while active development continues.

> [!IMPORTANT]
> Please consider starring the repository if you use or fork this project.

### List of features:

| Feature | Implemented |
|---------|-------------|
| Web Proxy | Yes |
| Browser-like UI | Yes |
| App player UI | Yes |
| Cloak features | Partially |
| Game Downloader | Yes |
| Quick Links | Yes |
| DuckDuckGo Search API | Yes |
| Apps & Games | Yes |
| Search Engine Switcher | Yes |
| Themes/Site Customization | Yes |

### Development & Building

#### Production:
```bash
git clone https://github.com/yashsurjuse/GhostProxy.git
cd GhostProxy/dogeub
npm i
npm run build
node server.js
```

#### Development:

```bash
git clone https://github.com/yashsurjuse/GhostProxy.git
cd GhostProxy/dogeub
npm i
npm run dev
```
